# 🎨 React Canvas Shape Manager

A React + TypeScript based canvas application that allows users to create, manage, and organize shapes as layers.

---

## 🚀 Features

- Add shapes: Circle, Square, Isosceles Triangle
- Choose color, size, and position (9-point grid)
- Real-time rendering using HTML5 Canvas API
- Layer system with delete functionality
- Undo / Redo support
- Persistent state using localStorage
- Smooth UI interactions and animations

---

## 🧠 Tech Stack

- React (Functional Components)
- TypeScript
- HTML5 Canvas API
- CSS (Custom styling, no UI libraries)
